# Frontend

该目录现在是“来整”微信小程序前端工程。

当前已具备的最小可运行结构：

1. 小程序入口文件：`app.js`、`app.json`、`app.wxss`、`sitemap.json`
2. 基础页面：首页、发起、我的活动、我的账单
3. 已接入本地图标资源：`img/brand-logo.jpg`

使用方式：

1. 用微信开发者工具直接打开 `frontend` 目录。
2. 使用当前 `project.config.json` 中的 `appid` 即可本地编译。
3. 若仅做界面调试，可保持未接后端接口状态运行。

当前页面以静态演示数据为主，后续再按 `../doc/` 下的产品与接口文档接入真实业务逻辑。
