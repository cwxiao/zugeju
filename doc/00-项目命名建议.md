# 小程序命名建议

## 推荐名称

### 组个局

推荐理由：

1. 语义非常直给，用户一眼就知道这是一个用来发起活动和组局的小程序。
2. 非常口语化，符合熟人之间真实的聊天语境，传播成本低。
3. 既适合游戏开黑，也适合聚餐、KTV、露营、钓鱼等线下活动。
4. 适合作为首版产品名，认知门槛低，不需要额外教育用户。

产品一句话解释：

一个帮朋友快速组局、查看回执、记录费用和公平结算的小程序。

## 备选名称

1. 聚友
2. 开个局
3. 朋友局
4. 局上见
5. 局友圈

## 当前建议

如果你希望产品名更直给、更容易让用户秒懂产品用途，我建议优先用：

组个局

如果你后续希望品牌感更强、名称更偏长期化运营，可以考虑备选：

聚友
