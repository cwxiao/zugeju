# 《来整》小程序 Q 版动画宣传视频制作指导

> 适用工具：剪映 AI 生成、即梦 AI、可灵 AI、Pika、Runway
> 视频时长建议：30～45 秒
> 风格定位：Q 版可爱、节奏轻快、年轻化

---

## 一、产品核心信息

| 项目 | 内容 |
|------|------|
| 小程序名 | 来整 |
| Slogan | 约人整活，一个按钮就搞定 |
| 目标用户 | 18～35 岁，喜欢社交、聚会、运动的年轻人 |
| 核心场景 | 约饭、约运动、约娱乐、AA 记账、朋友圈组局 |

---

## 三、分镜脚本（30 秒版）

### 🎬 Scene 1 — 痛点开场（0～4秒）

**画面**：Q 版主角（哈士奇拟人）躺在床上，手机屏幕亮着，微信消息弹个不停：
- "今晚谁约饭？"
- "有人打球吗？"
- "剧本杀还差 2 人！"

**文字浮层**：`每次组局…都像在打仗？`

**AI 提示词（即梦/可灵）**：
```
Q版卡通动画，可爱哈士奇拟人角色躺在床上，手机屏幕发出蓝光，
微信消息气泡不断弹出，表情烦恼又可爱，柔和的室内灯光，
皮克斯风格，3D卡通渲染，明亮色彩
```

---

### 🎬 Scene 2 — 产品亮相（4～8秒）

**画面**：手机屏幕特写，"来整"小程序图标弹出，闪光特效，Slogan 打字机效果出现：
**「约人整活，一个按钮就搞定」**

**文字浮层**：`来整 — 你的组局神器`

**剪映操作**：
- 用「贴纸」→ 搜索"手机界面"素材
- 叠加"来整"Logo（从小程序截图导入）
- 加「特效」→ 闪光/星星出现特效

---

### 🎬 Scene 3 — 创建活动（8～14秒）

**画面**：手指点击"开始整"，活动类型图标快速滚动（火锅🔥、K歌🎤、运动🏀、剧本杀🎭），最后停在"火锅"上，打勾✅

**文字浮层**：`50+ 活动类型，一键发起`

**AI 提示词**：
```
Q版3D动画，手机屏幕界面，手指点击按钮，各种活动图标像传送带一样滚动，
火锅、K歌、篮球、桌游图标，可爱卡通风格，鲜艳色彩，
动态模糊效果，高质量3D渲染
```

---

### 🎬 Scene 4 — 邀请好友（14～19秒）

**画面**：点"邀请"按钮，微信分享卡片飞出去，朋友们（不同动物拟人：金毛、猫咪、狼）纷纷点开接受，头像一个接一个跳进成员列表，带弹性动画

**文字浮层**：`微信一键分享，好友秒接受`

**AI 提示词**：
```
Q版卡通动画，一群可爱的动物拟人角色（狗、猫、狼）拿着手机，
微笑点击接受按钮，兴奋的表情，温暖的朋友圈氛围，
3D卡通风格，皮克斯风格，明亮温暖的光线
```

---

### 🎬 Scene 5 — 记账分账（19～25秒）

**画面**：火锅店场景，Q 版角色们举杯，画面切换到一个算盘/计算器的 Q 版形象在跳舞，账单自动分摊，每个人显示"应付 ¥46.5"，打勾✅

**文字浮层**：`消费记账，AA 自动算`

**AI 提示词**：
```
Q版3D动画，可爱的计算器卡通角色在跳舞，周围漂浮着¥46.5的数字气泡，
火锅餐桌场景，可爱动物角色们举杯，欢快氛围，
明亮的餐厅灯光，皮克斯风格，温暖色彩
```

---

### 🎬 Scene 6 — 性格报告（25～30秒）

**画面**：手机屏幕展示性格分析报告，"你是一只哈士奇🐕：社牛、夜猫子、气氛组"，生成海报，分享到朋友圈，点赞数飙升

**文字浮层**：`活动性格分析，朋友圈最靓的仔`

### 🎬 Scene 7 — 结尾号召（30～33秒）

**文字浮层**：`微信搜索「来整」立即体验`

## 四、角色设计指南（Q 版动物拟人）

| 动物 | 性格标签 | 视觉特征 |
|------|----------|----------|
| 哈士奇 | 社牛、夜猫子、气氛组 | 蓝眼睛、毛毛躁躁、笑容大 |
| 金毛 | 靠谱、人脉广、组局王 | 金色、温暖、竖大拇指 |
| 猫 | 佛系、精准出击、舒适区 | 慵懒、眯眼笑、优雅 |
| 狼 | 神秘、探险、野生能量 | 酷酷的、戴墨镜、运动风 |

**统一风格要求**：
- 头身比 1:1 或 1:2（Q 版经典比例）
- 大眼睛、小鼻子、圆润线条
- 配色明亮饱和（主色：橙 #FF7D21、蓝 #1198D4）

## 五、剪映 AI 生成操作步骤

### 方法 A：用「剪映 AI 生成」功能

1. 打开剪映 → 「AI 生成」→「图文成片」
2. 把下方的**旁白文案**粘贴进去
3. 选择「Q 版卡通」风格
4. 导出视频，再手动替换/精修不匹配的片段

### 方法 B：用「即梦 AI」生成素材片段，再导入剪映剪辑

1. 访问 [即梦 AI](https://jimeng.jianying.com)
2. 用上面的 AI 提示词逐个生成视频片段
3. 下载后导入剪映，加音乐、字幕、转场

### 方法 C：静态图 + 剪映「图片动效」

1. 用「Midjourney」或「即梦 AI 文生图」生成 Q 版场景图
2. 导入剪映，对每张图加「动效」→ 选择"轻微缩放"或"左右移动"
3. 叠加角色 PNG 素材，加弹性动画

## 七、音乐推荐

| 段落 | 音乐风格 | 剪映音乐库关键词 |
|------|----------|----------------|
| 开场痛点 | 轻松搞笑、稍快节奏 | 搜索：`搞笑 日常` `卡点 轻快` |
| 产品亮相 | 上升感、明亮 | 搜索：`品牌 宣传` `明亮 正能量` |
| 创建活动 | 欢快、卡点 | 搜索：`可爱 卡点` `Q版 动画` |
| 记账段落 | 轻松、有趣 | 搜索：`搞笑 计算` `轻快 BGM` |
| 结尾号召 | 高潮感、欢快 | 搜索：`欢快 结尾` `庆祝 彩带` |

## 八、字幕样式建议

- **字体**：思源黑体 Bold / 站酷快乐体（可爱感）
- **颜色**：白色字 + 橙色描边（#FF7D21，2px）
- **动画**：打字机效果（逐字出现）或弹性缩放出现
- **位置**：画面下方 1/4 处，避免遮挡主体

---

## 九、小程序码获取方式

1. 登录微信公众平台 → 登陆「来整」小程序后台
2. 「设置」→「基本设置」→「小程序码」
3. 下载 PNG（建议选 1280×1280 高清版）
4. 导入剪映，放在结尾画面中央
