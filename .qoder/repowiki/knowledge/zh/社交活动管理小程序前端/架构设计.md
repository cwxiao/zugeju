- 采用微信小程序标准架构，以 app.js/app.json 为全局入口，配置页面路由与全局状态（token/user）。
- 页面层（pages/）遵循四文件结构（.js/.json/.wxml/.wxss），通过 getApp() 访问全局登录态，利用 wx.navigateTo 进行页面跳转。
- 工具层（utils/）封装核心业务逻辑：request.js 实现多环境 API 切换与自动鉴权拦截；auth.js 处理微信一键登录流程；activity-insights.js 提供本地化的用户行为分析与人格报告生成算法。
- 数据层包含静态种子数据（data/ongoing.js, utils/activity-insights.js），用于前端展示与离线分析场景。
